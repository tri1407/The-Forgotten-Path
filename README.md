# 🎮 The Forgotten Path

**The Forgotten Path** is a strategic RPG focused on deep combat systems, resource management, and meaningful decision-making.

This project emphasizes system-driven gameplay rather than graphics or real-time mechanics. Every action has cost, consequence, and strategic weight.

---

## 🧠 Core Philosophy

- **Complex but Understandable Systems**  
  Mechanics are layered and interconnected, but always logically consistent.

- **Strategy Over Reflexes**  
  Combat rewards thinking, not reaction speed.

- **Quantifiable Design**  
  Every element in the game has measurable rules — damage, stamina, crafting cost, and success rates.

---

## ⚔️ Combat System

Weapons are more than stat boosts.

Each weapon defines its own set of actions:
- Strike
- Swing
- Pierce
- Shot

Defensive options include:
- Block
- Parry
- Dodge

Combat revolves around:
- Action economy
- Resource (stamina) management
- Risk vs reward decisions
- Tactical counterplay

Every choice matters.

---

## 🔨 Crafting System

Crafting converts resources into items using structured recipes.

Features:
- Validation before resource consumption
- Support for multiple craft amounts
- Clean separation between resources and items
- Expandable design for quality tiers and success rates

---

## 🎒 Inventory System

- Stackable resources
- Non-stackable variable items
- Designed to support future mechanics like enchantments and modifiers

---

## 🚀 Project Goals

- Build a scalable RPG system architecture
- Maintain clarity as complexity increases
- Create meaningful mechanics instead of decorative features
- Learn advanced system design through real implementation
